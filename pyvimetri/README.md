DESCRIPTION:
------------------------------------------------------------------------------------------------------------------
In 'pyvimetri' module, all Python tools which are used in ViMetRi-MAC/MAC 3.5b/065 project are included.
All functions in 'pymetri' module are encompassed in a particular submodule, in regards to its use in
the project.
